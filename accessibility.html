Inclusive Access

Accessibility Statement

UCfERI wants this website to be usable for students, entrepreneurs, staff, partners, and community members across devices, screen sizes, and access needs.

Accessibility Approach

We aim to provide clear navigation, readable typography, responsive layouts, descriptive links, visible focus states, and meaningful alternative text for key media where possible.

What We Are Improving

  • Consistent keyboard navigation across menus, forms, and buttons.
  • Clearer link destinations and fewer placeholder interactions.
  • Better colour consistency and contrast in shared interface components.
  • Accessible labels and status messages for website forms.

Known Limitations

Some older content, embedded media, and third-party platforms may not yet meet the same accessibility standard as the main site. UCfERI should continue reviewing images, video embeds, and downloadable resources as new content is added.

Feedback and Support

If you experience difficulty using any part of this site, contact cferi@univen.ac.za and describe the page or issue. Accessibility feedback should be treated as a support priority during ongoing website maintenance.

Need Assistance?

If a form or page does not work for you, UCfERI can still assist through direct email, phone, or WhatsApp channels listed on the site.
